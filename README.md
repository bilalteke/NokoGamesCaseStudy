# NokoGames — Arcade Idle Factory

An arcade idle factory game built with Unity 2022 LTS.

## APK
👉 [Download APK](../../releases/latest)

## Gameplay
Collect raw materials from the Spawner, deliver them to the Transformer, then take the output to the Sell Point to earn money. Hire an AI Worker to automate the whole process.

## Tech
- Unity 2022.3.24f1 LTS
- URP
- C# / State Pattern / SOLID
- DOTween / NavMesh / TextMeshPro / Joystick Pack
